# Deno webRTC Video Meet

Deno webRTC video meet and chat with native websocket.

> Requires Deno 1.12.2 or higher.

## Features

- Video Meet
- Chat
- Share Screen
- Room

## Run

```bash
deno run --allow-net --allow-read --unstable server.ts
```

open in http://localhost:3000/
